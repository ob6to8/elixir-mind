# first-order logic (FOL)

The standard formal language of objects, predicates, functions, and the
quantifiers ∀/∃, interpreted in structures via Tarskian satisfaction. Sound
and complete (Gödel 1929), compact, but only
[semidecidable](/glossary/semidecidability.md) and unable to pin down infinite
structures uniquely — the fully formal pole against which semiformal
knowledge representations define themselves. Canonically covered in
[FOL and OWL: fully formal knowledge representation](/knowledge-management/knowledge-representation/first-order-logic-and-owl.md).
