# nogood

An assumption set discovered to be jointly contradictory, recorded permanently
so that no [environment](/glossary/environment-atms.md) containing it is ever
explored again — the system's memory of dead ends, and what converts each
discovered contradiction into pruning of the whole search space. A nogood is
the TMS ancestor of the
[minimal inconsistent subset](/glossary/minimal-inconsistent-subset.md): both
name a smallest witness of joint contradiction, recorded rather than
rediscovered.
