# agent memory

The four kinds unpack as: short-term is the live context window itself; long-term holds facts and preferences that survive across sessions; procedural memory codifies workflows; working memory is the in-task scratchpad. This brain — markdown files under version control rather than a managed [vector database](/beliefs/glossary/vector-database.md) — is itself a file-based instance of the pattern.
